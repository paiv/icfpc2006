Source code for the Codex and contest tools.

This contains the entire source code for the Codex, and most of the tools we used to build the contest, including an ML to UM compiler, a UM assembler, puzzle generators for some of the problems, a graphical Antomaton simulator, etc. You can use these to build your own UM binaries.
